# Documentation

This directory contains long-form project docs. Common entries:

- `architecture.md` — system architecture overview
- `deployment.md` — production deployment runbook
- `security-audit-YYYY-MM-DD.md` — audit reports produced by `/run-security-audit`
- `adr/` — architecture decision records (one file per ADR)

For day-to-day reference, see [../CLAUDE.md](../CLAUDE.md) and [../.claude/commands.md](../.claude/commands.md).
